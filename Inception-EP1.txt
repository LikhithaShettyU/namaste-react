Library and framework difference
key difference between the two is inversion of control
when using libarary the control remains with developer who tells the application when to call library functions.
when using framework the control is reversed which means the framework tells the developer where code needs to be provided and calls it as it requires.

React known as React
React is JS library 
Name React was chosen because the library was designed to allow developers to react to change in state and data within an application 
update the user interface in declarative and effecient manner

Crossorigin
For loading external js files from diffrent origin than the hosting web page.
Allows the script to access resources from  the server hosting the script auch as making http requests or accesing data
